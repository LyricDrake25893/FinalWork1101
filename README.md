# Читай город

Программный комплекс для навигационных терминалов в торгово-развлекательных центрах, представляющий собой интерактивный каталог книжного маркета «Читай город».

## О проекте

Данный проект разработан с целью автоматизации обслуживания клиентов через электронные терминалы, расположенные на входе в ТЦ. Приложение позволяет детально ознакомиться с ассортиментом магазина и сформировать заказ.

Функциональность:
- Интуитивно понятный просмотр товаров в формате структурированных карточек
- нструменты сортировки и фильтрации
- Авторизация пользователей (Администратор, Менеджер, Клиент)
- Формирование и оформление заказов
- Автоматическая генерация индивидуальных талонов с кодом для получения книг
- Управление заказами для сотрудников
- REST API для интеграции
- Веб-приложение для просмотра каталога
